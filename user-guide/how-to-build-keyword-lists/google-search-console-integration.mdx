Get started by navigating to [Keyword discovery](https://app.keywordinsights.ai/keyword-discovery)

<img src="/images/CleanShot 2024-07-23 at 18.26.38@2x.png" alt="" />

1. Select Google search console.
2. Click Add to add a new Google account. 
3. Select a GSC property.

<Info>

Tip: You need to verify your search console property to gain access or must have access to a pre-verified account. Also select the sc-domain property to access all the features such as graphs etc. 

</Info>

Click 'Import'

<img src="/images/CleanShot 2024-07-23 at 18.31.03@2x.png" alt="" />

Depends on the number of keywords in the GSC property. it may take some time to import everything. 

<img src="/images/CleanShot 2024-07-23 at 18.32.38@2x.png" alt="" />

Once all the keywords are imported you can access the data. 

<img src="/images/CleanShot 2024-07-23 at 18.33.53@2x.png" alt="" />

1. Access filters. 
2. Graph showing your GSC data.
3. Shows number of clicks, impressions etc.
4. Keywords column.
5. Quick search volume fetch option.
6. Average search volume column.
