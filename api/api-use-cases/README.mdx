This is the hub for all use cases for the keyword insights public API. You can find payload examples, code snippets and lists of accepted parameters for the various endpoints.


You can access the Keyword Insights Public API here: https://api.keywordinsights.ai/apidocs/

The main use cases for the public API are:

* Clustering
* Content Brief

When using the public API, please make sure to first obtain a bearer token and use it with every request. The process how to do this is described [here](https://docs.keywordinsights.ai/~/changes/E5enC4TqwcsJQCwKLdMv/api/public-api-documentation).
